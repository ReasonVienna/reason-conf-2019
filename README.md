## Develop

Install all depedencies via

```sh
npm install
```

Run BuckleScript in one Shell

```sh
npm run re:start
```

Run Gatsby in another Shell

```sh
npm start
```

Visit `http://localhost:3000` in your browser

## Build

```sh
npm run re:build && npm run build
```
